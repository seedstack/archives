History and roadmap of the SeedStack project.
  • 2017-01-13 January

    SeedStack 16.11 "Kiwi"

    SeedStack 16.11 «Kiwi» includes the following features:

    • Completely new configuration system,
    • Netflix Feign add-on for writing Java HTTP clients easier,
    • Netflix Hystrix add-on (circuit breaker),
    • A new «tool mode» to launch various tools on application (like a configuration dump),
    • Various bug-fixes and improvements.

    This version requires Java 8.

  • 2016-11-17 November

    Paris Open Source Summit

    Modernize your applications using SeedStack & Docker at POSS, Paris / Devops track

  • 2016-09-22 September

    OW2con'16 Code to Product

    SeedStack : the lean development stack at the OW2 Conference, Paris. SeedStack joins the OW2 organization to reinforce application development tools.

  • 2016-07-31 July

    SeedStack 16.7 "Jasmine"

    SeedStack 16.7 «Jasmine» is a maintenance release.

  • 2016-07-11 July

    Ippon Technologies Event

    SeedStack Feature Tour at the Ippon Technologies event, Paris.

  • 2016-05-19 May

    Zenika NightClazz DDD

    Hands-on of SeedStack and its Business Framework at the NightClazz DDD meet’up at Zenika, Paris.

  • 2016-04-28 April

    SeedStack 16.4 "Iris"

    SeedStack 16.4 «Iris» includes the following features:

    • Fully Java 8 compatible,
    • Better Web integration,
    • Asynchronous REST resources and Servlets,
    • MQTT messaging add-on,
    • New Material design theme,
    • High-volume zoomable line chart, based on Dygraphs.
  • 2016-03-03 March

    SeedStack 15.11.3

    This intermediate version streamlines the project templates. Particularly, the Web project template has been refactored as a single-module project.

  • 2016-02-09 February

    SeedStack 15.11.2Details...

    This intermediate version includes fixed MongoDB, Solr, Neo4J and Redis modules which were not released correctly. It also include a newer version of w20-bridge.

  • 2016-01-21 January

    SeedStack 15.11.1Details...

    This intermediate version includes the following features and improvements:

    • Tightened security,
    • JAX-RS 2 support (through Jersey 2),
    • Swagger add-on,
    • Improved i18n performance,
    • Upgrade to the latest Angular 1.x version.
  • 2015-11-26 November

    SeedStack 15.11 "Hibiscus"Details...

    This version comes with a greatly improved add-on architecture and several new features:

    • MongoDB persistence,
    • Redis persistence,
    • Neo4J persistence,
    • Solr persistence,
    • Business framework improvements,
    • Cryptography support,
    • Injectable domain registry.
  • 2015-11-19 November

    Paris Open-Source Summit

    We will be presenting SeedStack at the Paris Open-Source Summit conference. Check the conference agenda for details.

  • 2015-07-28 July

    SeedStack 15.7 "Gingko"

    First public stable release of SeedStack.

  • 2015-02-01 February

    Creation of SeedStack OSS

    Creation of the GitHub organization. SeedStack is an Open-Source enterprise-class development stack derived from the former PSA Peugeot Citroën internal development stack. This internal version was used for two years by more than 20 projects.

  • 2013-10-14 October

    First internal release of Seed

    After 10 months of development with multiple beta releases, the first stable version of Seed, the Java framework, is released internally, at PSA Peugeot Citroën.

  • 2012-09-28 September

    First internal release of W20

    The first stable version of W20 is released. Since there was no Seed framework at the time, it was used with another, already existing backend framework.
